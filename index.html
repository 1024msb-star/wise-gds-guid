Gas Delivery System

GDS 압력 흐름 — Gas Cabinet → VMB/VMP → Process Tool

Primary·Reserve 듀얼 실린더에서 출발해 감압 후 VMB/VMP로 분배됩니다. VMB/VMP 헤더 위에 G2·T2·S2가 공통 1세트로 설치되고, 각 Stick에도 동일하게 반복 설치됩니다. 각 장비를 클릭하면 상세 설명이 표시됩니다.

설치 패턴 이해

실제 설치되는 3가지 조합 패턴

P810은 모든 패턴의 베이스입니다. 여기에 안전성·정밀도 요구 수준에 따라 Transmitter, Switch, 또는 둘 다가 추가됩니다. Transmitter를 쓴다고 Switch가 필요 없어지는 게 아니라, 둘은 안전 철학 자체가 다릅니다.

Transmitter를 쓰면 Switch가 필요 없어지는가 — 아니다
UPT863 (소프트웨어 기반 판단)
4~20mA 신호를 PLC가 받아 소프트웨어 로직으로 "임계치 초과 시 차단"을 판단. PLC·배선·소프트웨어 전체가 정상이어야 안전 기능이 성립
P840 (하드웨어 독립 동작)
압력 자체가 기계적으로 접점을 누르는 방식이라 PLC 로직과 무관하게 독립 동작. PLC가 다운돼도 마지막 안전판 역할 수행
고위험 유해가스의 안전 critical 구간은 "이중화"를 위해 Transmitter + Switch를 함께 두는 경우가 많습니다.

P810을 베이스로 항상 확보하고, UPT863과 P840을 함께 재고에 포함하면 패턴 1·2·3 어떤 고객사 구성에도 즉시 대응 가능합니다. Transmitter 도입이 늘어도 Switch 수요가 사라지지 않는다는 점이 핵심입니다.
보완 검토 사항
1. 캘리포니아 고객사의 패턴 1/2/3 분포 비중 확인
2. 안전 critical 구간(패턴 1) 비중이 높을수록 P840 동반 재고 필요
3. 정밀공정(패턴 2) 비중 확인 — UPT863 단독 수요 가늠
4. 위 표는 정성적 경향이며 실제 발주 데이터 축적 후 정량화 필요
