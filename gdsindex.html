GDS 압력 흐름 — Primary/Reserve 듀얼 실린더 구성

Primary·Reserve 실린더 각각에 G1 Gauge가 직결되고, S1 Switch는 두 라인이 합류한 지점에 공통으로 1개 설치됩니다. 각 장비를 클릭하면 상세 설명이 표시됩니다.

제품 역할 이해

P810 Gauge vs P840 Switch — 왜 둘 다 필요한가

두 제품 모두 원형 다이얼 indicator 타입입니다. P840은 아날로그 mechanical switch로, 전기 신호 출력은 가능하지만 자동 차단은 외부 제어 시스템이 필요합니다.

P840 아날로그 Mechanical Switch — 동작 원리와 한계
동작 원리
부르동관이 압력을 받아 변형 → 바늘이 움직임 → 바늘이 SET 포인터 위치의 기계적 접점을 물리적으로 누름 → 전기 접점 ON/OFF 신호 출력
신호 출력 후 무엇이 가능한가
가능 — 경보등·버저 ON/OFF, PLC 입력 신호, 릴레이 연결 시 외부 장치 제어
불가(단독) — ESO Valve 직접 자동 차단 → 외부 PLC·릴레이·솔레노이드 구성 필요
Primary/Reserve 듀얼 실린더에서 G1·S1 배치가 다른 이유
G1 (Gauge) — 실린더마다 개별 설치
Primary, Reserve 각각의 잔량을 개별로 알아야 자동전환 타이밍과 교체 계획을 세울 수 있음. 그래서 G1a, G1b 두 개가 각 실린더 Pigtail에 직결됨
S1 (Switch) — 합류 지점에 1개만
Switch의 목적은 "공급 라인 전체가 저압으로 떨어졌는지" 감지하는 것. 두 실린더가 Changeover Manifold에서 합류한 이후 공통 배관 한 곳에만 설치하면 충분 — 비용 절감 효과도 있음

3,000 psi 소량인 이유
  • 실린더 최대 충전압 3,000 psi 커버 필수
  • Primary/Reserve 구성 시 G1 2개 + S1 1개 = 3개
  • S1은 실린더마다 X, 합류 지점 1개로 충분
  • 교체 빈도 낮음 → 소량 재고로 충분
160 psi 주력인 이유
  • Regulator Outlet 20~150 psi 여유 있게 커버
  • VMB 헤더에 G2+S2 세트
  • 각 Stick에도 Gauge 추가 → 포인트 多
  • 교체·유지보수 빈도 높음 → 상시 재고 필수
